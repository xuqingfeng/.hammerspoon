### WindowResize

`ctrl + cmd + ⬅` - half left

`ctrl + cmd + ➡` - half right

`ctrl + cmd + ⬆` - half top

`ctrl + cmd + ⬇` - half bottom

`ctrl + cmd + 1` - top left

`ctrl + cmd + 2` - top right

`ctrl + cmd + 3` - bottom left

`ctrl + cmd + 4` - bottom right

`ctrl + cmd + C` - center

`ctrl + cmd + W` - one thrids center

`ctrl + cmd + F` - max screen
